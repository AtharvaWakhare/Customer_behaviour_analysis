# Customer_behaviour_analysis
Data analytics project showcasing customer behaviour analysis using pythons, sql and power bi.

Data Analytics Project
Overview

This project demonstrates an end-to-end Data Analytics workflow including data loading, exploratory data analysis (EDA), data cleaning, SQL analysis, and interactive dashboard creation.

The goal of the project is to extract meaningful insights from the dataset and present them through clear visualizations and dashboards to support data-driven decision making.

Dataset

The dataset used in this project contains structured data related to business transactions.

Key characteristics of the dataset include:

Customer information

Product details

Order and purchase data

Pricing and discount information

The dataset was analyzed to identify patterns, trends, and key performance indicators.

Tools & Technologies

The following tools were used during the project:

Python – Data loading, cleaning, and exploratory data analysis

Pandas – Data manipulation and transformation

Matplotlib / Seaborn – Data visualization

SQL Server – Writing queries for deeper data analysis

Power BI – Building interactive dashboards and reports

Jupyter Notebook – Running Python analysis

Project Workflow / Steps
1. Data Loading

Imported the dataset using Python.

Loaded the data into Jupyter Notebook using Pandas.

2. Exploratory Data Analysis (EDA)

Checked dataset structure and summary statistics.

Identified missing values and data types.

Analyzed distributions and relationships between variables.

3. Data Cleaning

Handled missing values.

Removed duplicates and incorrect records.

Standardized data formats and column names.

4. SQL Analysis

Loaded cleaned data into SQL Server.

Wrote SQL queries to extract key insights such as:

Total sales and revenue

Customer purchase behavior

Top performing products

Discount impact analysis

5. Power BI Dashboard

Connected Power BI to the processed dataset.

Built interactive dashboards to visualize key metrics.

Dashboard

The Power BI dashboard provides visual insights including:

Total sales and revenue overview

Customer purchasing trends

Product performance analysis

Sales distribution by categories

Impact of discounts on purchases

The dashboard allows users to interact with filters and slicers to explore the data.

Results & Insights

Key insights derived from the analysis include:

Identification of top-performing products

Understanding customer purchasing behavior

Analysis of sales patterns and trends

Insights into the effectiveness of discounts

These insights can help businesses improve decision-making and optimize their strategies.
